# Social asset pack (X-focused)

This folder contains reproducible scripts + generated assets for posting the Claude Lead System demo.

## Generate

```bash
python3 assets/demo/social/make_assets.py
```

## Outputs

- `assets/demo/demo.mp4` — ~30–40s X-native demo teaser
- `assets/demo/social/carousel/slide_*.png` — 4-slide carousel (1600×900)
- `assets/demo/social/thumbnails/*.png` — thumbnail variants

## Capture page (optional)

Open `assets/demo/social/landing/index.html` and screenshot/record it.
