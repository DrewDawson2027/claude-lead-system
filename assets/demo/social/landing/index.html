Everything your terminals were missing

Live Session Dashboard

See every Claude terminal at a glance — status, branch, tool counts (W/E/B/R), files touched, and last activity. No transcript parsing.

Pre-Edit Conflict Detection

The conflict-guard.sh hook warns before any Edit/Write if another session already touched that file — something Agent Teams fundamentally cannot do.

Cross-Terminal Messaging

Dispatch tasks with tell [session] to [task]. Messages arrive via PreToolUse inbox hooks — no context window pollution.

Autonomous Workers

Spawn headless claude -p workers. Track by PID, retrieve output, and terminate runaway tasks. Native tabs on macOS, Linux, and Windows.

Sequential Pipelines

Chain multi-step tasks: pipeline: task1, task2, task3 in [dir]. Each step completes before the next starts. Status tracked throughout.

Token Governance

The token-guard.py PreToolUse hook enforces a strict Tool Ladder, blocking expensive spawns when Grep/Read suffice. Hard cap: 5 agents/session.

Commands reference

/lead Boot the session dashboard — see all active + stale terminals
tell [session] to [task] Dispatch work to any active terminal via inbox hook
wake [session] with [msg] Wake an idle terminal + deliver message on next tool call
run [task] in [dir] Spawn a headless claude -p autonomous worker
pipeline: t1, t2, t3 in [dir] Run a sequential multi-step pipeline with status tracking
conflicts Cross-reference files_touched across all sessions
spawn terminal in [dir] Open a new interactive Claude Code terminal tab/pane
health check Validate all hooks, deps, and settings

How it works

01

Shell hooks fire on every tool call

PostToolUse, PreToolUse, SessionStart, and SessionEnd hooks run outside the context window — zero token cost.

02

Hooks enrich session JSON

Each terminal maintains a ~3 KB JSON file with tool_counts, files_touched, and recent_ops, updated on every invocation.

03

/lead reads JSON, not transcripts

The lead reads a few KB of JSON instead of MB of transcripts. Conflict detection, messaging, and worker control all flow from this single source of truth.
