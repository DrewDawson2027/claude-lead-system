# Demo Asset Pack

Use this script to produce social-proof assets for GitHub/launch posts.

## 1) Record terminal demo (2 minutes)

Recommended tools:
- `asciinema` + `agg` (terminal gif)
- or OBS (video)

Suggested flow:
1. Run installer
2. Open two Claude sessions
3. `/lead` boot dashboard
4. `tell ...` message dispatch
5. `conflicts` detection
6. `run ... in ...` worker spawn
7. `check worker ...` completed output

## 2) Asset filenames

- `demo.gif` (hero animation)
- `demo.mp4` (full walkthrough)
- `before-after.png` (problem vs solved)
