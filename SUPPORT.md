# Support

## Supported Versions

| Version | Status |
|---------|--------|
| Latest release | Supported |
| Previous release | Critical security fixes only (90 days) |
| Older | Unsupported |

Only the current release branch receives bug fixes, features, and security patches. Previous releases receive critical security fixes for 90 days after the next release.

## Getting Help

| Channel | Use For |
|---------|---------|
| [GitHub Issues](https://github.com/DrewDawson2027/claude-lead-system/issues) | Bug reports, install issues, feature requests, benchmark discrepancies |
| [GitHub Discussions](https://github.com/DrewDawson2027/claude-lead-system/discussions) | Questions, ideas, general discussion |
| [GitHub Security Advisory](https://github.com/DrewDawson2027/claude-lead-system/security/advisories/new) | Security vulnerabilities (private) |

## Response Expectations

| Type | Target Response |
|------|----------------|
| Security vulnerabilities | Acknowledge within 72 hours |
| Bug reports | Triaged within 7 days |
| Feature requests | Reviewed within 14 days |
| Questions | Best-effort community response |

These are targets, not guarantees. This is a solo-maintained open source project.

## Maintenance Cadence

- **Patch releases**: as needed for bug fixes and security patches
- **Minor releases**: quarterly, with new features and non-breaking improvements
- **Major releases**: as needed for breaking changes (rare)

## What's NOT Supported

- Custom forks or modifications
- Third-party integrations not documented in the repo
- Environment-specific issues (corporate proxy configs, custom shell setups, etc.)
- Claude Code itself — for Claude Code issues, see [claude.ai/code](https://claude.ai/code)

## Before Filing an Issue

1. Run `bash ~/.claude/hooks/health-check.sh` and include the output
2. Check [Troubleshooting](docs/TROUBLESHOOTING.md) for common fixes
3. Search [existing issues](https://github.com/DrewDawson2027/claude-lead-system/issues) for your problem
4. Use the appropriate [issue template](https://github.com/DrewDawson2027/claude-lead-system/issues/new/choose)
